# Modern Login & Registration Form

A beautiful and responsive login and registration form built with HTML, CSS, and JavaScript. This project features smooth animations, modern UI design, and user-friendly interactions.

## Features

- **Dual Functionality**: Seamlessly switch between Sign Up and Sign In modes
- **Password Visibility Toggle**: Show or hide password with eye icon
- **Smooth Animations**: Elegant transitions between different states
- **Modern UI Design**: Clean and professional interface with gradient effects
- **Responsive Layout**: Works perfectly on all device sizes
- **Form Validation**: User-friendly input fields with icons
- **Dynamic Content**: Context-aware text changes based on mode

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with animations and transitions
- **Vanilla JavaScript**: No external dependencies for functionality
- **Font Awesome**: Icon library for UI elements
- **Boxicons**: Additional icon support

## Project Structure

```
modern-login-registration-form/
├── index.html      # Main HTML structure
├── style.css       # Complete styling and animations
├── script.js       # JavaScript functionality
└── README.md       # Project documentation
```

## How It Works

1. **Sign Up Mode**: Users can register with username, email, and password
2. **Sign In Mode**: Users can login with email and password only
3. **Mode Switching**: Click buttons to toggle between registration and login
4. **Password Toggle**: Click the eye icon to show/hide password
5. **Visual Feedback**: Smooth animations and color transitions

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/ahamedenamul/modern-login-registration-form.git
   ```

2. Navigate to the project directory:
   ```bash
   cd modern-login-registration-form
   ```

3. Open `index.html` in your web browser
